🌳 Slektstre Live Demo

Interactive Family Tree Visualization with NetworkX

🎯 Hvordan bruke demo-en / How to use the demo:

🇳🇴 Norsk: Klikk på knappene nedenfor for å se forskjellige visualiseringer. Du kan zoome, panne og utforske slektstreet interaktivt.

🇬🇧 English: Click the buttons below to see different visualizations. You can zoom, pan, and explore the family tree interactively.

🎮 Interaktive funksjoner / Interactive Features:

  • 🖱️ Dra noder: Trykk og hold på en person med trackpad/mus, deretter dra for å flytte
  • 🔗 Elastisk bevegelse: Koblede personer følger med (10% bevegelse)
  • 🔍 Zoom: Scroll for å zoome inn/ut
  • 📱 Pan: Dra bakgrunnen for å flytte visningen
  • 📍 Hover: Hold musen over personer for detaljer

🎨 Visualiseringskonvensjoner / Visualization Conventions:

  • 🔵 Blå noder: Menn / 🔴 Røde noder: Kvinner
  • ➡️ Piler: Fra forelder til barn
  • 📏 Stiplede kanter: Ekteskap/partnerskap
  • 📋 Legend: Forklarer kanttyper

💡 Om denne demo-en / About this demo

Denne interaktive demo-en viser hvordan NetworkX kan brukes til å bygge og visualisere slektstrær. Du kan:

  • Zoome og panne: Bruk musen til å utforske slektstreet
  • Hover for info: Hold musen over personer for å se detaljer
  • Prøv forskjellige visninger: Klikk på knappene over for å se ulike visualiseringer

This interactive demo shows how NetworkX can be used to build and visualize family trees. You can:

  • Zoom and pan: Use your mouse to explore the family tree
  • Hover for info: Hover over people to see details
  • Try different views: Click the buttons above to see different visualizations
